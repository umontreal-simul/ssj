Detailed Description

This class simulates a specific stochastic activity network with 9 nodes and 13 links, taken from Elmaghraby (1977) and used again in L'Ecuyer and Lemieux (2000), "Variance Reduction via Lattice Rules".

The goal is to estimate the probability that the length of the longest path exceed a given constant x. This program is very specific to this example and uses a very naive way to compute the shortest path, by enumerating all six paths!
