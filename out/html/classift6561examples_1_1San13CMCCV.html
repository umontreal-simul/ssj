static void applyCV ( TallyStore  x,
TallyStore []  c,

)
static

Applies a vector of control variables to an estimator.

The tally x contains the observations used to estimate the output average while the array of tallies c contains the observations of the control variates. The third array, ec, contains the known expectations of the control variates. The length c and ec should be equal while x, and c should contain the same number of observations.

This method estimates the optimal vector of constants for the control variates, and prints information on the vector, and the variance with CV.

Parameters
xthe tally containing the observations for the output value.
cthe tally containing the observations of the control variates.
ecthe vector of expectations.
