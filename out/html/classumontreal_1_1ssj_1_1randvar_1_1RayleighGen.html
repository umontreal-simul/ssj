Detailed Description

This class implements random variate generators for the Rayleigh distribution.

Its density is

\[ f(x) = \left\{ \begin{array}{ll} \frac{(x-a)}{\beta^2}â€‰ e^{-(x-a)^2/2\beta^2} \quad & \mbox{ for } x \ge a \\ 0 & \mbox{ for } x < a, \end{array} \right. \]

where \(\beta> 0\).
