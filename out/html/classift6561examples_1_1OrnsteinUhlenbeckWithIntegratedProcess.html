Detailed Description

This class is custom-made for the CallableBond example.

It generates an Ornstein-Uhlenbeck process as in umontreal.ssj.stochprocess.OrnsteinUhlenbeckProcess, but it also computes the integral of the process, ${t_0}^{t} X() d$. The 2 x 2 correlation matrix for the process R(t) and its integral is decomposed with PCA.

◆ generatePath()

double [] generatePath ( )

Same as the method from the parent class, but also generates at each time step a stochastic variable that corresponds to the integral of the process over the time interval: ${t_{i-1}}^{t_{0}} X(t) dt$.

Those integrated values are not returned by this method, but are stored in memory and can be retrieved with getIntegratedPath().

Two random uniforms (from the inner RandomStream) are required to generate each time step instead of just one for the parent class.
