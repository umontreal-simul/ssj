Detailed Description

Implements an abstract class for density estimators that rely on the selection of a bandwidth \(h\) such as, e.g., DEHistogram or DEKernelDensity.

Furthermore, it provides basic methods such as to estimate the integrated variance (IV), for instance.

The IV of such estimators usually locally follow the model \(\textrm{IV}\approx C n^{-\beta}h^{-\delta}\) and their integrated square bias (ISB) is of the form \(\textrm{ISB}\approx B h^{\alpha}\), where \(n\) denotes the number of data points used to construct the density.

◆ computeIVandISBandMISE() [1/2]

double [] computeIVandISBandMISE ( double  estDensities[][])

This method estimates the IV, ISB, and MISE based on \(m\) realizations of the density estimator, which have previously been evaluated at the \(k\) points stored in evalPoints, and returns them in an array in this order.

The matrix estDensities has dimensions \(m\times k \), i.e. each row contains the evaluations of one density estimator.

The estimate for the IV is obtained from an estimate of the empirical IV computeIV(double[][]), the ISB is estimated by computeEstimatedISB(), and the MISE-estimate is obtained by their sum.

Parameters
estDensitiesthe \(m\times k\)matrix containing the results of evaluating \(m\) densities at \(k\) evaluation points.
Returns
estimates for the IV, the ISB, and the MISE.

◆ computeMISE() [1/2]

double computeMISE ( double  estDensities[][])

Computes the semi-empirical MISE for situations, where the true density is not known.

I.e., it takes the sum of the empirical IV computeIV(double[][]) and the estimated ISB computeEstimatedISB().

The estimate of the empirical IV is based on \(m\) realizations of the density estimator, which have previously been evaluated at the \(k\) points stored in evalPoints. The matrix estDensities has dimensions \(m\times k \), i.e. each row contains the evaluations of one density estimator.
