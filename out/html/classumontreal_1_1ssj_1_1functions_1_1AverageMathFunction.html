Detailed Description

Represents a function computing the average of several functions.

Let \(f_0(x), â€¦, f_{n-1}(x)\) be a set of \(n\) functions. This function represents the average

\[ f(x)=\frac{1}{n}\sum_{i=0}^{n-1} f_i(x). \]
