Detailed Description

Tools for Simple Monte Carlo and Quasi-Monte Carlo Experiments.

This package provides an interface to the LatNet Builder software to constructs quasi-Monte Carlo point sets and sequences of various types (lattices, polynomial lattices, digital nets, etc.) for arbitrary dimension and cardinality, various uniformity criteria, etc.
