Detailed Description

This class implements random variate generators for the triangular distribution.

Its density is

\[ f(x) = \left\{\begin{array}{ll} \frac{2(x-a)}{(b-a)(m-a)} & \mbox{for } a\le x\le m, \\ \frac{2(b-x)}{(b-a)(b-m)} & \mbox{ for } m\le x\le b, \\ 0 & \mbox{ elsewhere, } \end{array}\right. \tag{ftrian} \]

where \(a\le m\le b\) (see, e.g., [111] ).

The (non-static) nextDouble method simply calls inverseF on the distribution.
