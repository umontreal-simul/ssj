The following methods use the parameter ndig for the number of digits of absolute accuracy.

If the same methods are called without the ndig parameter, a default value of ndig = 15 will be used.

Computes the standard binormal distribution ( cdf2binormal ) with the method described in [47] , where ndig is the number of decimal digits of accuracy provided (ndig \(\le15\)). The code was translated from the Fortran program written by T. G. Donnelly and copyrighted by the ACM (see http://www.acm.org/pubs/copyright_policy/#Notice). The absolute error is expected to be smaller than \(10^{-d}\), where \(d=\) ndig.
