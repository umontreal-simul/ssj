Detailed Description

This class computes the same discrepancy for randomly shifted points of a set \(\mathcal{L}\) as given in eq.

( shift1 ) for class DiscShift1, but for the special case when the points are the nodes of an integration lattice [vHIC02a]  (eq. 16). It is given by

\[ [\mathcal{D}(\mathcal{L})]^2 = -1 + \frac{1}{n} \sum_{i=1}^n \prod_{r=1}^s \left[1 + \gamma_r^2 B_2(x_{ir})\right], \tag{shift1lat} \]

where \(n\) is the number of points of \(\mathcal{L}\), \(s\) is the dimension of the points, \(x_{ir}\) is the \(r\)-th coordinate of point \(i\), and the \(\gamma_r\) are arbitrary positive weights. \(B_2(x)\) is the Bernoulli polynomials [mABR70a]  (chap. 23) of degree \(2\). For a 1-dimensional lattice, the discrepancy becomes

\[ [\mathcal{D}(\mathcal{L})]^2 = \frac{1}{n} \sum_{i=1}^n B_2(x_i). \tag{shift1dim1lat} \]

Computing the discrepancy for a lattice is much faster than for a general point set.
