Detailed Description

This container class provides antithetic points.

That is, \(1 - u_{i,j}\) is returned in place of coordinate \(u_{i,j}\). To generate regular and antithetic variates with a point set p, e.g., for variance reduction, one can define an AntitheticPointSet object pa that contains p, and then generate the regular variates with p and the antithetic variates with pa.
