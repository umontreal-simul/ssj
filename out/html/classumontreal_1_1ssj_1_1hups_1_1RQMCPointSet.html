Detailed Description

This class is used for randomized quasi-Monte Carlo (RQMC) simulations [117], [118], [167], [168] .

The idea is to randomize a point set so that:

  • it retains its high uniformity when taken as a set and
  • each individual point is a random vector with the uniform distribution over \((0, 1)^s\).

A RQMC point set is one that satisfies these two conditions. One simple randomization that satisfies these conditions for an arbirary point set \(P_n\) is a random shift modulo 1 [34], [117], [194] : Generate a single point \(\mathbf{U}\) uniformly over \((0, 1)^s\) and add it to each point of \(P_n\), modulo 1, coordinate-wise. Another one is a random digital shift in base \(b\) [116], [118], [156] : generate again \(\mathbf{U}\) uniformly over \((0, 1)^s\), expand each of its coordinates in base \(b\), and add the digits, modulo \(b\), to the corresponding digits of each point of \(P_n\).
