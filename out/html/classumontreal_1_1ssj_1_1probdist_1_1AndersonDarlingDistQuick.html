Detailed Description

Extends the class AndersonDarlingDist for the Anderson–Darling distribution (see [6], [143], [200] ).

This class implements a version faster and more precise in the tails than class AndersonDarlingDist.

◆ cdf() [2/2]

static double cdf ( int  n,
double  x 
)
static

Computes the Anderson–Darling distribution function \(F_n(x)\) at \(x\) for sample size \(n\).

For \(0.2 < x < 5\), the asymptotic distribution \(F_{\infty}(x) = \lim_{n\to\infty} F_n(x)\) was first computed by numerical integration; then a linear correction \(O(1/n)\) obtained by simulation was added. For \(5 < x\), the Grace-Wood empirical approximation [70]  is used. For \(x < 0.2\), the Marsaglias’ approximation [151]  for \(n=\infty\) is used.

For \(n>6\), the method gives at least 3 decimal digits of precision except for small \(x\); for \(n \le6\), it gives at least 2 decimal digits of precision except for small \(x\). For \(n=1\), the exact formula \(F_1(x) = \sqrt{1 - 4e^{-x-1}}\), for \(x\ge\ln(4) - 1\), is used.
