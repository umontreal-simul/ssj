Detailed Description

This class implements digital sequences constructed from the Niederreiter sequence in base 2.

For details on these point sets, see [26] .

Constructor & Destructor Documentation

◆ NiedSequenceBase2()

NiedSequenceBase2 ( int  k,
int  w,
int  dim 
)

Constructs a new digital sequence in base 2 from the first \(n=2^k\) points of the Niederreiter sequence, with \(w\) output digits, in dim dimensions.

The generator matrices \(\mathbf{C}_j\) are \(w\times k\). Restrictions: \(0\le k\le30\), \(k\le w\), and dim \(\le318\).

Parameters
kthere will be 2^k points
wnumber of output digits
dimdimension of the point set
