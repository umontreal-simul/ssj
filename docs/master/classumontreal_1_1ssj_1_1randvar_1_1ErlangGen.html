Detailed Description

This class implements random variate generators for the Erlang distribution with parameters \(k > 0\) and \(\lambda> 0\).

This Erlang random variable is the sum of \(k\) exponentials with parameter \(\lambda\) and has mean \(k/\lambda\).

The (non-static) nextDouble method simply calls inverseF on the distribution.
