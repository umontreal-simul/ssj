Detailed Description

This class implements random variate generators having the Poisson distribution.

Its mass function is

\[ p(x) = \frac{e^{-\lambda} \lambda^x}{x!} \qquad\mbox{for } x=0,1,â€¦, \tag{fmass-Poisson} \]

where \(\lambda> 0\) is a real valued parameter equal to the mean.

No local copy of the parameter \(\lambda= \) lambda is maintained in this class. The (non-static) nextInt method simply calls inverseF on the distribution.
